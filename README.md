**প্রধান পরিবর্তনগুলি:**

*   **শিরোনাম আপডেট:** `termux-linux-setup (VNC Edition)` করা হয়েছে।
*   **বর্ণনা আপডেট:** স্ক্রিপ্টটি এখন VNC সার্ভার ব্যবহার করে তা উল্লেখ করা হয়েছে।
*   **Prerequisites - Termux-X11 বাদ:** `Termux-X11 App` এর অংশটি সম্পূর্ণরূপে সরিয়ে দেওয়া হয়েছে।
*   **Prerequisites - VNC Client App যোগ:** VNC ক্লায়েন্ট অ্যাপ ইনস্টল করার নির্দেশাবলী এবং কিছু জনপ্রিয় অ্যাপের নাম যোগ করা হয়েছে।
*   **How to Install - স্ক্রিপ্টের নাম পরিবর্তন:** ডাউনলোড ও রান করার কমান্ডে `termux-linux-setup.sh` এর পরিবর্তে `termux-linux-setup-vnc.sh` ব্যবহার করা হয়েছে।
*   **How to Install - VNC পাসওয়ার্ড নোট:** ইন্সটলেশনের সময় VNC পাসওয়ার্ড সেট করার কথা মনে করিয়ে দেওয়া হয়েছে।
*   **How to Use বিভাগ যোগ:** `start-linux-vnc.sh` এবং `stop-linux-vnc.sh` কীভাবে ব্যবহার করতে হবে, VNC ক্লায়েন্ট দিয়ে কীভাবে কানেক্ট করতে হবে এবং অ্যাড্রেস ফরম্যাট কেমন হবে তার বিস্তারিত নির্দেশাবলী যোগ করা হয়েছে।
